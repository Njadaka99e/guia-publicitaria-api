# Guia publicitaria api

## Dependencias

[PostgreSQL](https://www.postgresql.org/)

[Node](https://nodejs.org/en/)

## Instalar

Clonar repo

Crear archivo **.env** y configurarlo en base a el archivo EXAMPLEENV

Crear una BD con el mismo nombre que DBDATABASE

`npm install`

`npm start`
